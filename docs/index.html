Everything you need

All the features you'd want from a Roblox launcher, built in and ready to go.

⚡

Performance Presets

One-click FastFlag bundles — Default, Balanced, and Max Performance — instantly tune Roblox's renderer for your hardware.

🔁

Auto-Rejoin

Disconnected? Shadowstrap detects when Roblox exits and automatically relaunches it, dropping you back into the same server.

🎮

Discord Rich Presence

Show your friends exactly which game you're playing, which server you're on, and how long you've been in-session.

🖼️

Mod Support

Replace cursors, fonts, sounds — including a custom death sound picker — by dropping files into the Modifications folder.

🪟

Multi-Instance

Run multiple Roblox windows simultaneously. Shadowstrap closes the singleton mutex so a second (or third) instance can start.

🚀

Priority Boost

Elevate Roblox's CPU scheduling priority to Above Normal, High, or Realtime so it gets the CPU time it needs.

🧹

RAM Cleaner

Trim other processes' working sets before Roblox launches, giving the game as much free physical RAM as possible.

🌐

Hone.gg Integration

Automatically launch the Hone.gg network optimizer before every Roblox session to reduce ping and stabilize your connection.

🏳️

Fast Flag Editor

Browse, edit, and manage any Roblox FastFlag directly from the built-in JSON editor without digging through config files.

Common questions

Is this malware?

No. The entire source code is open and auditable right here on GitHub. Downloads come only from this repository.

Will I get banned?

No. Shadowstrap only replaces the launcher/bootstrapper. It does not interact with the game client the way exploits do.

Does it work with Studio?

Yes. Studio launching is supported, though some player-only features (Discord RPC, activity tracking) won't apply.

How do I uninstall?

Open Shadowstrap → Settings → Shadowstrap tab → Uninstall. This restores the default Roblox bootstrapper.
