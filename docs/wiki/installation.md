# Installation

## Requirements

- Windows 10 or 11 (64-bit)
- [.NET 6 Desktop Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/6.0) — you'll be prompted to install it automatically if it's missing

## Steps

1. Go to the [Releases page](https://github.com/binx-ux/shadowstrap/releases/latest)
2. Download `Shadowstrap.exe`
3. Run it

That's it. Shadowstrap installs itself, registers as the Roblox protocol handler, and adds itself to the Start Menu. The next time you click **Play** on any Roblox game, Shadowstrap will handle the launch.

## SmartScreen Warning

Windows may show a SmartScreen popup on first run because Shadowstrap is not yet widely known to Windows Defender. This does **not** mean it's malicious.

To dismiss it:
1. Click **More info**
2. Click **Run anyway**

See [SmartScreen Warning](smartscreen.md) for more detail.
